# CF-Workers-GitAutoSync
Cloudflare Workers 定时批量拉取远程文件并自动同步至 GitHub 仓库工具
## 项目简介
### 核心功能
1. KV 可视化管理后台：密码登录、增删改查同步任务、在线预览配置
2. Cron 定时自动执行批量同步任务，支持自定义定时周期
3. 远程文件拉取 + JSON 自动格式化美化
4. 批量文件一键提交推送至 GitHub 指定仓库分支
5. HTTP 接口手动触发同步，密钥鉴权防滥用
6. 自定义 404 兜底页面，全链路异常捕获，解决 Workers Error 1101 超时报错
7. 双触发模式：定时 scheduled 事件 / 网页 Fetch 请求触发

### 访问地址说明
- 可视化管理后台：`你的Worker域名/ckadmin`
- 手动同步触发接口：`你的Worker域名/run?key=自定义RUN_KEY`

## 一、前置准备（4步一次性操作）
### 1. 生成 GitHub 个人访问令牌 GH_TOKEN
1. 登录 GitHub，打开令牌创建页面：https://github.com/settings/tokens/new
2. 表单配置：
   - Note：`CF-Workers同步工具`
   - Expiration：`No expiration`（永久有效）
   - Scopes：仅勾选顶部 `repo` 全部分类权限
3. 滑至页面底部点击 `Generate token`，完成账号二次验证
4. **立刻复制**生成的 `ghp_xxxx` 字符串，页面关闭后无法重新查看，丢失需重新生成令牌

### 2. 整理6项环境变量（部署Worker时填入）
| 环境变量名 | 填写说明 |
| ---- | ---- |
| GH_TOKEN | 上一步获取的 ghp_ 开头 GitHub 令牌 |
| GH_USER | GitHub 主页用户名（地址 https://github.com/用户名 中后半段） |
| GH_REPO | 目标存储文件的仓库名称 |
| GH_BRANCH | 仓库提交分支，绝大多数项目填 `main` |
| RUN_KEY | 自定义随机字符串，/run 接口鉴权密钥（示例：Abc123456789） |
| ADMIN_PWD | 后台 `/ckadmin` 登录密码，自定义 |

### 3. 创建 Cloudflare KV 持久化命名空间
用于存储同步任务配置，永久保存任务列表
1. 登录 Cloudflare 控制台：https://dash.cloudflare.com/
2. 左侧导航栏：`Workers & Pages` → `KV`
3. 点击「创建命名空间」，自定义命名（推荐：`git-sync-config`）
4. 创建完成后复制命名空间ID，后续绑定Worker使用

### 4. 账号要求
Cloudflare 免费账号即可部署，免费额度完全满足日常定时同步需求，无需付费升级。

## 二、创建 Cloudflare Worker 项目
1. 返回 `Workers & Pages` 首页，点击「创建应用程序」
2. 选择「创建 Worker」
3. 自定义项目名称（示例：`git-auto-sync`），点击「部署」
4. 等待初始化完成，点击「快速编辑」进入在线代码编辑器

## 三、部署核心代码操作
1. 在代码编辑器内全选并删除默认生成的全部模板代码
2. 将项目完整业务代码复制粘贴至编辑器内
3. 点击页面右上角「保存并部署」完成代码上传

## 四、绑定 KV 命名空间 & 配置环境变量
### 4.1 绑定KV存储
1. Worker详情页 → 设置 → 变量
2. 滚动至「KV命名空间绑定」，点击添加绑定
   - 变量名称固定填写：`SYNC_KV`
   - KV命名空间选择上一步创建的 `git-sync-config`
3. 保存绑定配置

### 4.2 添加环境变量（密钥）
同页面「环境变量」板块，依次添加前文表格6个变量，**全部勾选加密**，防止密钥明文泄露。

## 五、配置定时 Cron 触发器（自动同步）
1. Worker详情页 → 触发器 → 添加Cron触发器
2. 填写Cron表达式自定义同步周期：
   - 每小时执行：`0 * * * *`
   - 每天凌晨2点执行：`0 2 * * *`
   - 每6小时执行：`0 */6 * * *`
3. 保存触发器，到达设定时间自动拉取文件并同步GitHub

## 六、功能使用教程
### 6.1 后台管理面板
访问 `域名/ckadmin`，输入配置的 `ADMIN_PWD` 登录
支持操作：
- 添加同步任务：填写远程文件URL、仓库存储路径
- 编辑/删除已有同步任务
- 查看任务列表、上次同步日志

### 6.2 手动一键触发同步
浏览器/工具访问接口，替换参数为你自定义的RUN_KEY
