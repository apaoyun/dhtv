## 📦 一键部署
> [!IMPORTANT]
> 部署前请准备好 Cloudflare 账号，免费额度足够日常使用

1. 登录 Cloudflare 后台
2. 创建 Workers 项目，粘贴代码
3. 新建 KV 存储空间并绑定
4. 修改配置密码，保存发布即可
